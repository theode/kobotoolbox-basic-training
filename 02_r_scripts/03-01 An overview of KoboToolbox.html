1 What is KoboToolbox?

  • KoboToolbox is a suite of applications providing an open source data collection tool for working in “challenging” environments.
  • It is designed to address many of the shortcomings of mobile data collection.
  • KoboToolbox lends reliability, accuracy, speed, data quality, analysis and cost effectiveness to mobile data collection.
  • KoboToolbox was developed by the Harvard Humanitarian Initiative in association with Brigham and Women’s Hospital and USAID.
  • Works for data collection offline or online on virtually any device.
  • Its main features are free to use.

2 KoboToolbox Features

2.1 Web Form Builder

  • Allows for the designing of very basic to fairly complex electronic forms or questionnaires.
  • Has 25 quantitative and qualitative question types to choose from (text, numeric, date and time, single- or multi-select, ranges, audio, video, geospatial, and much more).
  • Many useful features can be included in forms to ensure high quality data is collected efficiently, such as skip logic, validation, and even language translation.

2.2 Question Library

  • Saves questions as templates which can be reused in future forms.

2.3 Data collection via mobile app or Web browser

  • Allows data to be collected online or offline
